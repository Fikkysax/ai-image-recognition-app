# AI Image Recognition App

This is a free AI image recognition web app built with Python, TensorFlow, and Gradio.

## What the app does

The user uploads an image, and the AI predicts the top 5 possible objects in the image.

## Tools used

- Python
- TensorFlow / Keras
- MobileNetV2
- NumPy
- Pillow
- Gradio
- GitHub

## How to run the app

bash
python -m pip install -r requirements.txt
python app.py
