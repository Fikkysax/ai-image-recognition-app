# AI Image Recognition App

A free AI image recognition web app built with Python, TensorFlow/Keras, MobileNetV2, and Gradio.

## Live Demo

[Open the live app](https://huggingface.co/spaces/Fikkysax/ai_image_recognition_app)

## What the app does

This app allows a user to upload an image. The AI model then predicts the top 5 possible objects in the image and shows the confidence level for each prediction.

## Features

- Upload an image
- Predict the main object in the image
- Show top 5 predictions
- Display confidence scores
- Simple web interface
- Free local AI model
- Deployed online with Hugging Face Spaces

## Tools used

- Python
- TensorFlow/Keras
- MobileNetV2
- NumPy
- Pillow
- Gradio
- GitHub
- Hugging Face Spaces
